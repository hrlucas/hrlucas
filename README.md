## Sobre mim

Sou estudante técnico em **Desenvolvimento de Sistemas** e atuo como **Jovem Aprendiz na área de TI**, unindo aprendizado técnico, prática profissional e desenvolvimento de projetos próprios.

Tenho foco em desenvolvimento **full stack**, criação de **APIs**, sistemas web, automação, integrações, dashboards e soluções aplicadas a problemas reais. Busco construir projetos com boa organização, documentação clara, arquitetura escalável e atenção à experiência do usuário.

Meu interesse principal está em transformar ideias em sistemas funcionais, bem estruturados e preparados para evolução.

## Projetos públicos

Meus repositórios públicos funcionam como um portfólio técnico: cada projeto foi criado para praticar ou demonstrar uma competência específica, como desenvolvimento de APIs, construção de dashboards, integração com serviços externos, automação, persistência de dados, arquitetura modular ou interfaces responsivas.

### Projetos em destaque

| Projeto | Descrição | Stack principal |
|---|---|---|
| [HR CultivaTech System](https://github.com/hrlucas/hr-cultivatech-system) | Plataforma web para monitoramento de hortas inteligentes, com dashboard interativo, simulação de dados em tempo real, autenticação por perfil e arquitetura preparada para evolução. | React, TypeScript, Vite, TailwindCSS |
| [HR WhatsApp API](https://github.com/hrlucas/hr-whatsapp-api) | API REST + WebSocket para controle de sessão, envio de mensagens, contatos, grupos, conversas, rótulos, canais e documentação Swagger/OpenAPI. | Node.js, Express, WebSocket, whatsapp-web.js |
| [HR YouTube Transcriber](https://github.com/hrlucas/hr-youtube-transcriber) | Aplicação web para baixar mídia do YouTube, transcrever áudio localmente e exportar resultados em `.txt` e `.md`, com revisão opcional por LLM local. | Python, FastAPI, yt-dlp, faster-whisper, Ollama |
| [HR Intelbras LPR Webhook](https://github.com/hrlucas/hr-intelbras-lpr-webhook) | Sistema para recebimento de eventos LPR de câmera Intelbras, persistência de leituras, painel web, API REST, fallback local e integração opcional com WhatsApp. | Python, Flask, SQLAlchemy, PostgreSQL, SQLite, Node.js |
| [HR Auction System](https://github.com/hrlucas/hr-auction-system) | Sistema desktop para gerenciamento de produtos de leilão, com cadastro, listagem, venda, consulta de vendidos e persistência em banco de dados. | Java, Java Swing, MySQL, JDBC |
| [HR BarberDesk System](https://github.com/hrlucas/hr-barberdesk-system) | Sistema desktop para operação de barbearia, com cadastro inicial, autenticação, gestão de serviços, barbeiros, agenda e histórico persistido. | Java, Java Swing, MySQL |
